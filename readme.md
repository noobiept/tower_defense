Keyboard Shortcuts
==================


- `1` -- Select `basic tower`.
- `2` -- Select `fast tower`.
- `3` -- Select `rocket tower`.
- `4` -- select `frost tower`.
- `5` -- Select `anti-air tower`.
- `6` -- Select `bash tower`.
- `n` -- `N`ext wave.
- `u` -- `U`pgrade selected tower.
- `s` -- `S`ell selected tower.
