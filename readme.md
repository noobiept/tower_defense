Maps
====

- `easy`:
    - 20 waves
    - 1 lane

- `medium`:
    - 50 waves
    - 2 lanes

- `hard`:
    - 60 waves
    - 2 lanes
    - with some obstacles


Keyboard Shortcuts
==================


- `1` -- Select `basic tower`.
- `2` -- Select `fast tower`.
- `3` -- Select `rocket tower`.
- `4` -- select `frost tower`.
- `5` -- Select `anti-air tower`.
- `6` -- Select `bash tower`.
- `n` -- `N`ext wave.
- `u` -- `U`pgrade selected tower.
- `s` -- `S`ell selected tower.
